# Resume API

A REST API for an AI Resume Builder built with Node.js + Express.

## How to Run

```bash
npm install
node app.js
```

Server runs on: `http://localhost:3000`

## Routes

### Auth
- POST `/api/auth/register` тАФ Create account
- POST `/api/auth/login` тАФ Login
- POST `/api/auth/logout` тАФ Logout
- POST `/api/auth/forgot-password` тАФ Forgot password
- POST `/api/auth/reset-password` тАФ Reset password

### Users
- GET `/api/users/me` тАФ Get profile
- PUT `/api/users/me` тАФ Update profile
- DELETE `/api/users/me` тАФ Delete account

### Documents
- GET `/api/documents` тАФ List documents
- POST `/api/documents` тАФ Create document
- POST `/api/documents/import` тАФ Import document
- GET `/api/documents/:id` тАФ Get one document
- PUT `/api/documents/:id` тАФ Update document
- POST `/api/documents/:id/duplicate` тАФ Duplicate document
- DELETE `/api/documents/:id` тАФ Delete document

### Sections
- POST `/api/documents/:id/sections` тАФ Add section
- PATCH `/api/documents/:id/sections/:sectionId` тАФ Edit section
- DELETE `/api/documents/:id/sections/:sectionId` тАФ Remove section
- POST `/api/documents/:id/sections/:sectionId/items` тАФ Add item
- PATCH `/api/documents/:id/sections/:sectionId/items/:itemId` тАФ Edit item
- DELETE `/api/documents/:id/sections/:sectionId/items/:itemId` тАФ Remove item

### Versions
- GET `/api/documents/:id/versions` тАФ List versions
- POST `/api/documents/:id/versions` тАФ Save version
- POST `/api/documents/:id/versions/:versionId/restore` тАФ Restore version

### Templates
- GET `/api/templates` тАФ List templates
- GET `/api/templates/:id` тАФ Get one template

### AI
- POST `/api/ai/bullets` тАФ Generate bullet points
- POST `/api/ai/summary` тАФ Generate summary
- POST `/api/ai/rewrite` тАФ Rewrite text
- POST `/api/ai/prompt` тАФ Apply instruction

### Applications
- GET `/api/applications` тАФ List applications
- POST `/api/applications` тАФ Log application
- PATCH `/api/applications/:id` тАФ Update status
- DELETE `/api/applications/:id` тАФ Remove application
