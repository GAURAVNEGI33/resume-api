# 📄 Resume API

A backend project built with **Node.js** and **Express.js** to learn REST APIs, routing, controllers, models, and middleware using an industry-style project structure.

> 🚀 This project is being developed step by step during my backend learning journey.

---

## 📂 Project Structure

```text
resume-api/
│
├── backup/                  # Previous CRUD implementation
├── controllers/             # Request handling logic
├── middleware/              # Custom middleware
├── models/                  # Data access layer
├── routes/                  # API routes
│   ├── index.js
│   └── documentRoutes.js
├── app.js                   # Entry point
├── package.json
├── package-lock.json
└── .gitignore
```
